# NetSage Android (MVP)

目标：做一个可上架的安卓应用，核心是“网络故障智能诊断”。

## MVP 功能
1. 上传/粘贴网络日志
2. 输出 Top3 根因 + 修复建议
3. 导出排障报告（文本）

## 目录
- `docs/`：PRD、路线图、上架清单
- `android-app/`：Android 客户端（Kotlin + Compose）
- `backend/`：诊断 API（FastAPI）

## 本周目标
- 完成安卓端骨架 + 首页/结果页静态流程
- 完成后端 `health` 与 `diagnose` 占位 API
- 跑通端到端联调（mock 数据）
