# Play 审核备注模板（中英双语）

## 中文
本应用为网络故障诊断工具。测试时无需注册账号。
进入首页后粘贴日志文本，点击“开始诊断”即可看到 Top3 诊断结果。
点击“复制报告”可将诊断摘要复制到剪贴板。

## English
This app is a network troubleshooting assistant.
No account is required.
Paste log text on the first screen and tap "开始诊断" to view Top 3 causes.
Tap "复制报告" to copy the generated report to clipboard.
